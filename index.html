Mini tutorial

Para utilizar esta aplicación, sigue los siguientes pasos:

  1. Descarga el repositorio desde GitHub.
  2. Instala las dependencias con el comando "npm install".
  3. Inicia el servidor con el comando "npm start".
  4. Abre tu navegador y accede a la dirección http://localhost:3000.
  5. Explora la aplicación y disfruta.
